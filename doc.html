Project Overview

Symme is a secure communication application designed to provide users with private, encrypted, and anonymous messaging and calling.

Features

  • End-to-end message encryption & decryption
  • Disappearing messages
  • Encrypted audio & video calls with rerouting
  • Support for images, files, and attachments
  • Anonymous sign-in using unique UID
  • No screenshots, no screen recording
  • QR code scanning for secure ID exchange
  • Custom themes and modes

Architecture / System Design

Tech stack: Flutter, Dart, Firebase

System flow includes anonymous login, QR code exchange, encrypted storage via Firestore, and secure audio/video call routing.

Installation / Setup

git clone https://github.com/your-repo/symme.git
cd symme
flutter pub get
flutter run

Configuration

Set up Firebase Authentication, Firestore rules, and secure environment variables.

Usage

  1. Sign in anonymously (UID auto-generated).
  2. Scan QR code to connect with another user.
  3. Send encrypted messages, files, or attachments.
  4. Start secure audio or video calls.

API Reference

Example endpoints:

  • POST /messages/send – Encrypt and send a message
  • GET /messages/{conversationId} – Fetch messages
  • POST /calls/initiate – Start a secure call

Troubleshooting & FAQ

  • App not connecting to Firebase: Verify API keys.
  • Messages not disappearing: Check Firestore TTL rules.
  • Screenshots possible: Confirm device restrictions.
